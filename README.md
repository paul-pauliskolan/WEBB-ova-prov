# Assignment – Hur webben fungerar

I denna uppgift ska du skapa en informationswebbsida om hur webben fungerar.

Du ska använda HTML, CSS och lite JavaScript.

## Innehåll på sidan

Din sida ska innehålla text som förklarar:

- Vem som uppfann webben och när
- Vad ett internetprotokoll är
- Skillnaden mellan HTTP och HTTPS
- Vad GDPR handlar om
- Vad upphovsrätt innebär
- Vad responsiv design är

## HTML-krav

Din sida ska innehålla:

- semantiska element (header, nav, main, section, article, footer)
- minst en bild
- minst en länk
- ett formulär
- alt-attribut på bilder

## CSS-krav

Du ska använda:

- minst en klass
- minst ett id
- Flexbox
- Grid
- margin och padding
- responsiv design med media queries

## JavaScript

Du ska skapa en knapp som visar mer information på sidan.

Du måste använda:

- document.getElementById()
- console.log()

## Inlämning

1. Klona repot
2. Arbeta i VS Code
3. Commit och push
